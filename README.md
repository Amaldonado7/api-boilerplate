# TP2 Apps Mobile | News API

React Native project with axios and expo that uses the [news api](https://newsapi.org/docs/endpoints/top-headlines) to show top headlines. 

# Usage

```
# Install dependencies
npm install
```

```
# Run app
expo start
```
